# BPP Brand Voice

## Summary
Calm operator who knows what they're doing and respects people's time. Direct, confident, friendly. Entrepreneur to entrepreneur. Use data when it helps someone decide. Never get technical unless asked. Every message has a point, a next step, and a clean close.

## Tone Rules

**Direct and confident.** State what's happening and what you need. Don't apologize for clarity.
- Use: "Here's what we finished and what's next."
- Avoid: "Just checking in..." / "Sorry to bother you..."

**Professional but approachable.** Clean writing, clear structure, friendly energy.
- Use: "Quick update and next steps below."
- Avoid: "Per our previous correspondence..."

**Entrepreneur-friendly, not corporate.** Talk about outcomes, time, money, and systems. Plain language only.
- Use: "This will stop leads from slipping through the cracks."
- Avoid: "Synergies" / "Stakeholders" / "Operationalize"

**Data-informed, not technical.** Numbers when they help. Summarize the meaning, not the math.
- Use: "You're averaging ~12 leads/month. Raising follow-up from 30% to 70% produces more sales without more ad spend."
- Avoid: SQL talk, tool internals, CRM mechanics

## Hard Rules
- Never use "leverage" as a verb. Use: apply, build on, plug into, streamline.
- Never use em dashes. Use periods, commas, colons, or restructure the sentence.
- Never use filler: "Just wanted to...", "I think...", "I hope this email finds you well"
- Numbers like a business owner speaks: "~$3.8K/month" not "approximately three thousand"
- "We will" not "We can"

## Default Message Structure
1. Purpose line (one sentence)
2. Key info (2-5 bullets max)
3. Ask / next step (clear action, deadline if relevant)
4. Close (simple)

## Closings
- "Appreciate it."
- "Thanks, talk soon."
- "Once I get this, I'll send the next draft."
- "If anything looks off, call it out and we'll adjust."
